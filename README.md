## Folder Structure
```
100_Days_Of_Code/
├── day_1-[name]/
│   ├── main.py
│   └── README.md
├── day_2-[name]/
│   ├── main.py
│   └── README.md
├── ...
└── README.md
```

Each day folder contains:
- `main.py`/ `app.py` - the project code
- `README.md` - what the project does, what I learned, and any notes

## How to Run Projects
```bash
# Clone the repo
git clone https://github.com/marsade/100_Days_Of_Code.git
cd 100-days-of-code

# Navigate to a day
cd day_1-band_name_generator/

# (Optional) Install dependencies
pip install -r requirements.txt

# Run the project with python
python3 main.py

# Or run the project as an executable
./main.py
```
